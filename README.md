# Aadhaar Enrolment Data Analysis

This project is based on Aadhaar enrolment data.
It shows how new Aadhaar enrolments change across states, districts, age groups, and time.

This project is made for learning data analysis using Python and Power BI.

## Project Contents
- Python Code (Jupyter Notebook)
- Dataset 
- Power BI Dashboard
- Project Report

## Tools Used
- Python
- Jupyter Notebook
- Pandas
- Matplotlib
- Power BI

## Analysis Done
- State-wise enrolment analysis
- District-wise enrolment analysis
- Age-wise enrolment analysis

## Purpose
- To understand Aadhaar enrolment pattern
- To practice data analysis
- To create dashboard using Power BI

## Purpose
This project is created for learning data analysis and visualization.
